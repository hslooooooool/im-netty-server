# 消息服务SDK NETTY版本
包含Netty版本SDK和SpringBoot使用案例
## SDK说明
im-sdk-netty项目为SDK源码。

如果使用IDEA，则将此module加入artifact->build后即可生成项目用SDK jar包。

## Demo说明
Demo采用SpringBoot 2.1.x与Kotlin 1.3.x编写。工具为IntelliJ IDEA 2019.3。
